# 微信小程序项目开发指南（纯前端版）

## 技术栈
- **框架**：微信小程序原生框架
- **语言**：TypeScript
- **样式**：WXSS（支持 rpx）
- **数据存储**：wx.setStorageSync / wx.getStorageSync（本地缓存）
- **后端接口**：无需调用任何外部 API

## 目录结构
miniprogram/
├── pages/ # 页面文件夹
├── components/ # 自定义组件
├── utils/ # 工具函数
├── data/ # 本地模拟数据（如需要）
├── app.js # 小程序入口逻辑
├── app.json # 全局配置
├── app.wxss # 全局样式
└── project.config.json # 项目配置文件

## 代码规范
- 页面尺寸使用 **rpx** 实现自适应
- 所有数据持久化使用 **wx.setStorageSync**（同步方法，简单可靠）
- 组件命名采用 **kebab-case**（如 `user-card`）
- 工具函数放在 `utils/` 目录，模块化导出

## 常用命令
- **启动开发**：微信开发者工具 -> 打开项目
- **预览**：点击工具栏「预览」生成二维码
- **清除缓存**：开发者工具 -> 工具 -> 清除缓存（用于测试）

## 开发注意事项（纯前端重点）
1. **数据存储**：所有用户数据都存本地，使用 Storage API
2. **首次使用**：需要初始化默认数据（如空账单列表）
3. **数据导出**：如需导出数据，可生成 JSON 文件并保存到本地
4. **包体积限制**：单个包不超过 2MB，图片资源尽量使用网络图片或压缩
5. **无需配置 request 合法域名**（因为不调用任何接口）

## 设计规范
- 主色调：[#769C4A]
- 字体大小：标题 36rpx，正文 28rpx，辅助文字 24rpx
- 圆角：卡片 16rpx，按钮 8rpx，小元素 4rpx
